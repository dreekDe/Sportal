# Sportal
IT-Talents Season 14 - Final project

Web based Java Application for sports news. 

TECHNOLOGIES:
- Java 11
- Spring Boot
- Spring Rest
- Hibernate
- Lombok
- BCrypt
- MySQL
- Maven
- IntelliJ IDEA
- GiT
- Postman
- MySQL Workbench

FUNCTIONALITIES:
Guest:    
- View all articles
- Filter top 5 articles
- Show article details
- Search for specific article/s by given title or category
- Register

User:
- Login/Logout
- Edit and delete user
- Add comment to article
- Reply to comment
- Like and dislike comment
- Download image

Admin:
- Upload and delete image
- Add and delete category
- Add, edit and delete article
- Find user/s
- Delete user
